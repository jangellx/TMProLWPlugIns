Installing the SDK

Before you install the SpaceWare SDK 1.0 for Windows 95 and Windows NT, you:

  • Need to know the drive letter of the drive from which you will load the SpaceWare SDK
  • Must have the SpaceWare driver version 8.0 or later installed on your computer

Follow these steps to install the SDK: 

  1. Insert the SpaceWare software media into the CD-ROM drive.
  2. Click on Start > Run. Make sure the command line e:\setup.exe appears in the edit box and click on OK. If you install from another drive, substitute that drive letter for e. 
  3. Follow the Installation program prompts.

 
Changes in this Release

This release of the SpaceWare SDK contains the following changes from previous releases:

  • Only versions of the library compiled with the Microsoft Visual C++ 4.2B compiler are provided.
  • The SpaceWare Input library, siapp, is now produced as a DLL, with source code provided to rebuild the static library portion.
  • The tuning for the device has changed internal to the library. Any applications using a previous version of the SDK may need to change the tuning constants to maintain the correct feel.
  • SiDispatch is provided for backward compatibility only. It is not recommended that you use this function.
  • SiSetTypeMask is obsolete, but a stubbed version is provided for backwards compatibility.
  • Two new demos, sbtest32 and sbtstmfc, are included.
  • All Help documentation is in HTML format.

Release notes


SpaceWare DLLs

As part of the SDK, two Dynamic Link Libraries (DLLs), siappdll.dll and siappdld.dll, are installed in your Windows system (or system32 for Windows NT) directory. These files must be in the Windows system directory in order for the Spaceball to work properly. Spacetec gives developers full license to distribute these files. These same DLLs are in the following directories:

lib\uapi\bin\siappdll\<platform>\debug\siappdld.dll

the debugging version of the Spaceball DLL

lib\uapi\bin\siappdll\<platform>\release\siappdll.dll

the release version of the Spaceball DLL

SpaceWare Libraries

The SpaceWare library consists of two parts: the Input Library and the Math Library. These libraries should be built separately, combined into a single library, and then linked into your application. The Input Library and Math Library were generated using the Microsoft Visual C++ compiler, versions 4.2B.

There are two versions of this SpaceWare library: siappst.lib and siappmt.lib. The first has been compiled for single-threaded applications and the second for multithreaded applications. They are located in the hardware platform-specific directory in the %SDK%\lib\siapp\bin\<platform> directory. To add SpaceWare support to an application you must link one of the SpaceWare libraries (siappst.lib or siappmt.lib) to the application.

Note:
You must also link the standard winmm.lib into the application.
