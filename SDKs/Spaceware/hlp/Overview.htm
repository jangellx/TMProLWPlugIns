SpaceWare Software Developer's Kit Overview

The SpaceWare SDK provides the resources you, the developer, need to add SpaceWare support to application software. The SDK includes support for Windows NT and Windows 95. See Supported Platforms for the specific platforms and compilers supported by this SDK.

The following parts are provided:

  • SpaceWare Libraries, including the SpaceWare Input Library and the SpaceWare Math Library
  • All include files required to integrate with the SpaceWare libraries
  • Installation for the libraries, documentation, demos, and source code
  • This documentation explaining how to build and use the SpaceWare libraries and demos
  • Several programming examples
      CityFly, showing eyepoint control similar to a typical walk-through application
      Widget World, showing object control similar to a typical CAD application
      sbtest32, a Win32 program that displays the Spaceball data
      sbtstmfc a Win32 program that displays the Spaceball data using MFC
