HyperVoxels Properties and Morph Mixer Properties are simple generic plug-ins for applying and opening HyperVoxels and Morph Mixer in Lightwave.

To install, just follow these steps:

  • If any copies of Layout are running, quit them now.
  • Create a new directory in your Lightwave Plugins directory called TMPro and copy HyperVoxelProperites.p and MorphMixerProperties.p into it.
  • Load Layout
  • Select Add Plug-ins.
  • Find and add HyperVoxelProperties.p and MorphMixerProperties.p. Layout should report that a total of three plug-ins have been added.
  • Quit Layout. This will update your Lightwave config and ensure that everything works properly.
  • The next time you run Layout, you can launch HyperVoxel Properties and Morph Mixer Propertiesfrom the Generics pop-up in the Plug-in Options Panel, or assign it to a key or button on the interface with Configure Keys and Configure Menus. Please refer to your Lightwave documentation for more information. The plug-ins will be prefixed with TM_P under the Plugins heading of the Configure Keys and Configure Menus panels.

HyperVoxels Properties

When activated, HyperVoxels Properties will check to see if HyperVoxels is already in the scene. If not, it will be added and it's interface will be opened; if so, then the existing HyperVoxel instance's interface will be opened. This gives you quick access to HyperVoxels with a simple keystroke or button press


Morph Mixer Properties
Morph Mixer Properties (Apply)

Similar to HyperVoxels Properties, when activated, Morph Mixer Properties (Apply) will check to see if Morph Mixer is already to the selected object. If not, it will be added and it's interface will be opened; if so, then the existing HyperVoxel instance's interface will be opened.

Morph Mixer Properties acts identically to Morph Mixer Properties (Apply), but will not add the plug-in if it isn't already there. This is useful if you simply want to open the Morph Mixer interface without inadvertantly adding the plug-in to the object.
